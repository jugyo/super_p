## Installation

Add this line to your application's Gemfile:

    gem 'super_p'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install super_p

## Usage

    sp object

specify the color like this:

    SuperP.color = "\e[45;35;5m" # blinking
    sp object
